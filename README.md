# hubot-hello

Hello world

See [`src/hello.coffee`](src/hello.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-hello --save`

Then add **hubot-hello** to your `external-scripts.json`:

```json
[
  "hubot-hello"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
